# task7
This project demonstrates how to extract and analyze sales data from a small SQLite database using Python. It provides a quick overview of sales performance per product.
Features:
Run SQL queries in Python to get total quantity sold and total revenue per product
Load data into Pandas for easy manipulation
Display results using print statements
Visualize data with a simple bar chart using Matplotlib
Save the chart for reporting purposes (sales_chart.png)
Lightweight workflow using Jupyter Notebook or .py script
Tools Used: Python (sqlite3, pandas, matplotlib), SQLite (built-in)
Deliverables:
Python script / Jupyter Notebook with SQL queries
Printed sales summary
Bar chart visualization
Outcome & Learning:
Learned to integrate SQL queries within Python
Practiced data aggregation and basic analytics
Created simple visualizations to communicate sales insights quickly and effectively
Gained hands-on experience with SQLite databases and Python data workflows
